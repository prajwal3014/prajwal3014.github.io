Img tag in HTML

In websites or webpages you always interact with images and pictures. As images and pictures increases the readability of your websites. So while making website, if you want to add images you can add them with the help of tag. Below is the syntax of < img > tag
< img src="image_name.extension" width="20%" height="200px"  alt="Alternate text " >

Here, src, width, height and alt are attributes of tag.
⦁ src : It specifies the path of the image.
⦁ width : It determines the width of image in terms of %(percentage) or px(pixels)
⦁ height : It determines the height of image in terms of %(percentage) or px(pixels)
⦁ alt : If for some reason image cannot be displayed, this attribute then specifies an alternate text for image.
Note : We can set height and width of the image with the help of CSS also. We’ll discuss about it later in the upcoming blogs.

Anchor tag in HTML

Many times in websites you see there are some links where you click over there and it navigates you to some other webpages, images, any file etc.
So, if you want to make such links to a webpage, file etc. you can do it with the help of anchor tags.

Syntax :
< a href = “your link…” > Click here < /a >
Here, a tag defines the hyperlink and href attribute specifies the link URL where you want to visit.
Between the anchor tags a message is wriiten (in syntax : Click here) to tell that by clicking on it there is a link.
