HTML Elements

 An HTML element can be defined as :
<tag name>
Content ……..
………………
………………
</tag name>

<tagname> is your starting tag, followed by the content and </tagname> is your ending tag.

Example :
<h1>Welcome to Uniquedu</h1>
<p>Web development</p>


Basic Elements in HTML structure :


• <!DOCTYPE html> : This element defines that the document is an HTML document
• <html> : This element contains the HTML document.
• <head> : This element contains information/metadata about the HTML page
• <title> : This element defines a title for the HTML page and it can be seen in the browser's title bar or in the page's tab.
• <body> : This element defines the document's body. Your maximum working will be inside this element.
