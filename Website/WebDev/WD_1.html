Introduction

Greetings of the day everyone, this is Harshit Srivastava and I have done certifications in web development courses. So, in this series of blogs, I am going to give you a full tutorial on Web development.

Before starting with the Web development tutorial, you must know what is a website and how websites work. Talking about the dictionary meaning, a website is a set or collection of webpages, which are related to each other and they are placed under a single or common domain name.

Now, from the above meaning, generally a question arises that what is meaning of domain name. So, domain name is the address of a website that people type in the browser to visit that particular website. For example, when in the browser you type www.youtube.com, it navigates you to the YouTube page. Here youtube.com is the domain name.

Now coming to another point how website works. Basically web development is divided into two categories.
1. Front End Development
2. Back End Development

We’ll briefly discuss about both the categories and this will clear that how website works.
1. Front End Development :

From the name “Front End Development”, we can get that it is development of the front i.e., layout, structure, animations, interactions and other things that we see on the website.

Dictionary terms, Front-end web development is the practice of converting data to a graphical interface, through the use of HTML, CSS, JavaScript and Bootstrap, so that users can view and interact with that data.

2. Back End Development :

From the name “Back End Development”, we can get that it is development of the backend part that we can’t see but its role is important. For example, when you sign up in a website with your emailid and password, your details gets saved in the backend database like MongoDB through Nodejs + Express. And again when you come to login on that page with your details, you have to just type emailid and password, and after that in the backend part, your details are being fetched through Express+Nodejs from the MongoDB database, and after verifying you get logined to the page. This was an example to demonstrate that how backend works. Don’t get afraid of the terms like MongoDB, Nodejs, Express. These are backend languages and they’ll be covered soon in the upcoming tutorials.

I hope you got now how website works.

So, this is all about the Introduction of Web Development. Thank you to be so patient. I hope you gained a brief knowledge about what is web development and how website works.
