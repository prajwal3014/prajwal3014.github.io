Basic Syntax

Greetings of the day everyone, this is Prajwal Sharma. I hope you all liked my previous blog. So, in this blog, we will talk about “Basic Syntax of C language” and write our first program.

Coming to the first program in C language, First learn the basic syntax :
• Header files : In every program we have to include header files because it becomes easy for the compiler to understand the program.
• Main Function : By reading the name you can understand that it is the main part of program, Every executable thing is done inside main function.
• Open Curly Braces & Close Curly Braces : It will tell compiler that all the statements inside them is together part of main function. It is called function body.
• In the function body, we have to write our code for desired output.
• At last, the return statement is important because after execution it will tell us that our code is successful or not.
Now after you understand the basic things, just open your Dev C++, Press Ctrl+N for new tab and Write the following code :
#include<stdio.h>
int main()
{
printf("Hello Guys, Welcome to UniquEdu...!");
return 0;
}
After writing the code, Save it with extension “.c”, By default Dev C++ save the code with extension “.cpp” which is for C++ but we have to save it for C that’s why take care of it.
When it is saved then we have to first compile it, for compiling click on button shown in image. Its shortcut is F9.

After compiling the program, if your program is all correct with no error you can see at the bottom of your screen a box.

If there is an error in code you will see like this :

The red line will tell you where error must be, There you can see that before return statement a semicolon is missing so it created an error.
By looking this error, I want everyone to note a point that a semicolon or terminator is very important in C language and in every statement when you want to show that this statement is terminated here then you have to put a semicolon.
Now your code is compiled, this is the time to run it. Now press the run button shown in image :

Shortcut for it is F10.
When you pressed it, you can see an exe window and you can see the message written there.

Look closely you can see a statement here “return value 0”, What is it? In basic syntax I told you that it will tell us about our code is successful or not, so look here “return value 0” means code is successful but if there is value other than 0 then the code is not successfully executed.
At last, I want to tell you that if you want to compile and run at a same time then use this button. Shortcut is F11.

So, That’s all for this blog. I hope you all will execute the program. Thank you to be so patient.
