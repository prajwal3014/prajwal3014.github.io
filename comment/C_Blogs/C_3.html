There are many softwares for programming C language, but in my opinion Top three are Dev C++, Turbo C++, Visual Code.
In this whole series we use Dev C++ for writing our code.
Here are the steps to install :
• Go to this link download Dev C++.
• Click on download Button shown on the page.

• Wait until it gets downloaded.
• When downloaded open it and just click next and next, set the path and finally click on install or finish button to install it.
• Now you can see a shortcut on your desktop screen, just double click on it to open.
• When it opens choose your purpose language and you can see like this on your screen.

• Now press ctrl+N to open a new source file for coding like this.

• Now use the editor for writing your code.
